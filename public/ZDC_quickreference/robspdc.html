"ROB'S PDC" COMMAND REFERENCE

.rpdc{x} {yy} {zz}

{x} = the initial altitude in thousands.

  • values of 3 thru 7 are supported.
  • substitute "vs" for "Climb Via Sid."
  • (note that there is NO SPACE between "rpdc" and the digit or the "vs".)

{yy} = the ID of the controller handling departures.

  • use the Controller List within VRC to determine the appropriate two-character alphanumeric ID.
  • if departure is offline, substitute a lower-case "u" ("uncontrolled") and OMIT THE SPACE BEFORE IT.

{zz} = expected runway for departure.

EXAMPLES

.rpdc3 1j 30 initial altitude = 3,000, departure controller id = 1j, expect runway 30 for departure.
.rpdcvs 32 28 climb via SID, departure controller id = 32, expect runway 28 for departure.
.rpdc5u 1 initial altitude = 5,000, departure offline, expect runway 1 for departure.
.rpdcvsu 15R climb via SID, departure offline, expect runway 15R for departure.
